## 배열 제어하기 
#### 권장 시간 : 10분 
#### 권장 시간복잡도 : O(NlogN)
---
정수 배열을 하나 받을때, 배열의 중복값을 제거하고 배열 데이터를 내림차순으로 정렬해서 반환하는 solution()함수를 구현하시오.

**제약 조건**
- 정수 배열의 길이는 $2 \le N \le 1000$ 이다
- 정수 배열의 각 데이터 값의 범위는 $-100,000 \le x \le 100,000$ 입니다.


**입출력의 예**

| 입력                    | 출력                    |
| --------------------- | --------------------- |
| [4, 2, 2, 1, 3, 4]      | [4, 3, 2, 1]      |
| [2, 1, 1, 3, 2, 5, 4] | [5, 4, 3, 2, 1] |
